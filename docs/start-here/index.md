# Learning paths

NKP brings together Kubernetes, cloud native operations, infrastructure, and AI
workloads. You do not need to master every component before using the platform,
but shared terminology and mental models make cross-team design easier.

These optional paths are organized by topic and role. Experienced readers can
skip directly to [Architecture](../architecture/index.md),
[Installation](../install/index.md), or [Upgrades](../upgrades/index.md).
